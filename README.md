#DIVE
중고 거래 사이트<br>

### 1. 기술/구조

* 웹 표준
    * `HTML5`
    * `CSS3`
    * `JavaScript`
    * `jQuery`
* Tomcat v9.0 사용
* DBMS - `Oracle 11g`
* SPRING Framework
* `JSP`, `EL`
* `MyBatis`
* Api 사용
    * `jQuery`, `aJax`, `JSON` 파싱
* `SPRING MVC 구조`
* `RESTful Api 구조`
    * 서버 <--> 클라이언트 통신을 위한 REST 인터페이스 구현
